# react-native-ios-app-clips

iOS 14 App Clips in React Native

## Installation

```sh
npm install react-native-ios-app-clips
```

## Usage

```js
import IosAppClips from "react-native-ios-app-clips";

// ...

const result = await IosAppClips.multiply(3, 7);
```
